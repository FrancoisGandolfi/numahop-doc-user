    Unité documentaire​

    Les unités documentaires sont accessibles depuis > Unités documentaires. La liste des unités est affichée dans la colonne de gauche. Lorsque vous cliquez pour sélectionner une unité, son contenu s’affiche.

    Détail de l’unité documentaire

    La zone de travail sur les notices est accessible via le bouton   Le détail des informations est présenté sous forme d’onglets.

    Créer une unité documentaire

    L’unité documentaire est une entité qui doit toujours être rattachée à un lot.

    Pour y accéder il faut cliquer sur Documents puis sélectionner Unités documentaires.

    Cliquer sur l’icône + pour créer une nouvelle unité documentaire et remplir le libellé  de l’unité documentaire.

    Renseigner les champs de la fiche de l’unité documentaire en indiquant à quel lot et à quel projet elle doit être rattachée. Renseigner également l’Identifiant PGCN. Il doit correspondre au nom du document qui doit être livrer, tel qu’il est nommé dans le FTP.

    Renseigner le champ Radical : celui-ci doit être identique au champ «Identifiant PGCN».

     

    Indiquer si le document doit être archivé au CINES et s’il peut être diffusé.

    A cette étape, vous pouvez configurer la diffusion sur Internet Archive ou Omeka. Si cela est le cas, renseigner le nom des collections d’Internet Archive et d’Omeka dans lesquels le document doit être placé après traitement.

    Une fois la fiche remplie, cliquez sur Enregistrer.
