# Créer un lot

Un lot est associé à un seul projet et permet une répartition
intellectuelle des unités documentaires d’un projet.

Vous pouvez créer des lots associés aux projets dans le menu <img
src="http://www.numahop.fr/wp-content/uploads/2022/10/documents-numaHOP.png"
class="alignnone wp-image-134" decoding="async" width="22"
height="26" /> &gt; Lots.

Cliquer sur l’icone
<img src="http://www.numahop.fr/wp-content/uploads/2022/10/Ajouter.png"
class="alignnone wp-image-202 size-full" loading="lazy" decoding="async"
width="47" height="42"
alt="icone &quot;plus&quot; permettant les ajouts dans NumaHOP" /> pour
créer un nouveau lot.

Renseigner les champs de la fiche « Lot » en indiquant à quel projet il
doit être rattaché.

Si des champs ne sont pas remplis, par défaut, ils prendront les valeurs
indiquées dans le projet auquel le lot est rattaché.

 

 puis cliquer sur  « enregistrer ». 

Le nom du nouveau lot apparaîtra dans la colonne de gauche.

Tant que le lot n’est pas démarré, il est possible d’ajouter
<img src="http://www.numahop.fr/wp-content/uploads/2022/10/Ajouter.png"
class="alignnone wp-image-202" loading="lazy" decoding="async"
width="29" height="26" /> ou de supprimer <img
src="http://www.numahop.fr/wp-content/uploads/2022/10/supprimer.png"
class="alignnone wp-image-200" loading="lazy" decoding="async"
width="28" height="27" /> des documents du lot. Une  
fois le workflow lancé, le périmètre du lot est figé.  
Dans tous les cas, il est possible de sélectionner des unités
documentaires et d’exporter les données du tableau au format CSV <img
src="http://www.numahop.fr/wp-content/uploads/2022/10/exporter-1.png"
class="alignnone wp-image-201" loading="lazy" decoding="async"
width="24" height="24" />

Il est possible de consulter l’unité documentaire ou la livraison en
cliquant sur <img
src="http://www.numahop.fr/wp-content/uploads/2022/10/Consulter.png"
class="alignnone wp-image-203" loading="lazy" decoding="async"
width="28" height="24" />
