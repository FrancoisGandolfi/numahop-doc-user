# <Icon icon={["fa", "cart-flatbed-suitcase"]} /> Batch Management

A batch is associated with a single project and provides an intellectual grouping of documentary units within that project.

You can create batches associated with projects via the menu <img
src="http://www.numahop.fr/wp-content/uploads/2022/10/documents-numaHOP.png"
class="alignnone wp-image-134" decoding="async" width="22"
height="26" /> &gt; Batches.

Click the icon
<img src="http://www.numahop.fr/wp-content/uploads/2022/10/Ajouter.png"
class="alignnone wp-image-202 size-full" loading="lazy" decoding="async"
width="47" height="42"
alt="plus icon for adding items in NumaHOP" /> to create a new batch.

Fill in the fields of the "Batch" form, indicating which project it should be linked to.

If any fields are left blank, they will default to the values specified in the associated project.

Then click "Save".

The name of the new batch will appear in the left column.

As long as the batch has not been started, it is possible to add
<img src="http://www.numahop.fr/wp-content/uploads/2022/10/Ajouter.png"
class="alignnone wp-image-202" loading="lazy" decoding="async"
width="29" height="26" /> or remove <img
src="http://www.numahop.fr/wp-content/uploads/2022/10/supprimer.png"
class="alignnone wp-image-200" loading="lazy" decoding="async"
width="28" height="27" /> documents from the batch. Once the workflow has been started, the scope of the batch is fixed.

In all cases, it is possible to select documentary units and export the table data in CSV format <img
src="http://www.numahop.fr/wp-content/uploads/2022/10/exporter-1.png"
class="alignnone wp-image-201" loading="lazy" decoding="async"
width="24" height="24" />.

It is possible to view the documentary unit or delivery by clicking on <img
src="http://www.numahop.fr/wp-content/uploads/2022/10/Consulter.png"
class="alignnone wp-image-203" loading="lazy" decoding="async"
width="28" height="24" />.
