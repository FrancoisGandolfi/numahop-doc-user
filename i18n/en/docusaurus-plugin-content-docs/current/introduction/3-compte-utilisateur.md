# User Account <Icon icon={["fa", "user"]} />

## Login & Logout

Logging in requires a username, or *login*, and a password. The **username** is determined by the naming rules set by the administrator. The **password** is provided during registration for the application.

:::info
The password must contain at least twelve characters, including one uppercase letter, one lowercase letter, one number, and one special character.
:::

The possible actions for the logged-in user's account are accessible via the dropdown menu in the NumaHop header bar: **First Name** <Icon icon={["fa", "caret-down"]} />.

- <Icon icon={["fa", "wrench"]} /> **Profile**;
- <Icon icon={["fa", "lock"]} /> **Password**;
- <Icon icon={["fa", "arrow-right-from-bracket"]} /> **Logout**.

---

### User Profile

The details of the logged-in user's profile are accessible via the dropdown menu in the NumaHop header bar. The user can modify certain information about themselves, depending on their permissions.

- Click on the user's name: **First Name** <Icon icon={["fa", "caret-down"]} />;
- Select the option: <Icon icon={["fa", "wrench"]} /> **Profile**.
- Modify the available fields.

|      Field       | Status      | Details                     |                                                                     _See also_ |

|    **Login**     | Required    | Free text                   |                                                                                   |
|     **Name**     | Required    | Free text                   |                                                                                   |
|  **First Name**  | Required    | Free text                   |                                                                                   |
|   **Active**     | Default     | Yes / No                    |                                                                                   |
|  **Position**    | Optional    | Free text                   |                                                                                   |
|   **Phone**      | Optional    | Free text                   |                                                                                   |

### Password Change

#### By the User

A user can change their own password directly via the dropdown menu in the NumaHop header bar.

- Click on the user's name: **First Name** <Icon icon={["fa", "caret-down"]} />;
- Select the option: <Icon icon={["fa", "lock"]} /> **Password**.

If a user forgets their password, they can reset it without involving an administrator from the login screen by clicking on **"Forgot Password?"**. The user then enters their login, and a new password is sent to the email address registered on their account.

#### By the Administrator

In case of loss, a new password can be generated by an administrator.

- From the <Icon icon={["fa", "user"]} /> **Users** menu, select the relevant user;
- In the dropdown menu <Icon icon={["fa", "bars"]} /> **Other Actions**, choose **"New Password"**;
- A new password is generated and displayed in the right panel <Icon icon={["fa", "circle-info"]} /> **Information**.
