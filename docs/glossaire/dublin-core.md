# Dublin Core

Le _Dublin Core_ est un vocabulaire du web sémantique utilisé pour
exprimer les données dans un modèle RDFa. Il a pour objectif de fournir
un socle commun d’éléments descriptifs suffisamment structuré pour
permettre une interopérabilité minimale entre des systèmes indépendants.
